# Documentation

This folder contains technical documentation for the Research Team Manager plugin.

## Available Documentation

### [Field Mapping Documentation](FIELD-MAPPING.md)
Comprehensive guide for mapping fields from legacy WordPress sites to the Research Team Manager plugin during import processes. Includes:
- Field mapping tables for ACF to RTM field conversion
- New fields available in the plugin
- Automatic field population logic
- Member status and type mapping

### [Custom Sorting Documentation](SORTING-DOCUMENTATION.md)
Detailed guide for implementing custom sorting options in Query Loop blocks. Covers:
- Available sorting methods (start date, end date, custom order, priority sorting)
- Implementation instructions for Gutenberg blocks
- Code examples and troubleshooting
- Advanced sorting scenarios
